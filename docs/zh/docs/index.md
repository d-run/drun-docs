# d.run 让算力更自由

!!! tip

    d.run = DaoCloud Runs Intelligence [立即体验 d.run](https://console.d.run/){ .md-button }
    
    d.run 是 DaoCloud 自研的 AIGC 综合性算力运维和运营平台，
    整合云原生能力，构建智能应用体系，化算力为“算利”。

    - 管理：负责用户访问控制、权限管理、审计日志以及 d.run 个性化设置等
