# 月账单

月账单提供按月汇总的消费统计信息，并支持按区域、按产品、按账号等维度进行拆分，同时支持账单导出，帮助用户从账期角度了解费用结构。

    ![monthly-bills](images/monthly-bills-03.png)

## 字段说明

进入 **账单管理** 页面，点击 **月账单** ，可查看月账单列表。列表按账期展示账户每个月的消费情况，支持点击时间框来选择 **账期** 。

| 显示字段 | 说明 |
| --- | --- |
| 账期 | 月度消费周期 |
| 原价 | 未扣除优惠前的总费用 |
| 代金券抵扣 | 该账期由代金券抵扣的总金额 |
| 现金支付 | 扣除优惠后实际支付金额 |
| 详情 | 点击后查看该账期的多维度统计 |

## 详情

点击 **详情** 图标，进入该账期的多维度费用详情。账单详情包含三个维度：

    ![monthly-bills](images/monthly-bills-02.png)

1. 按区域统计

* 按资源所在的地域展示费用拆分。

| 字段 | 说明 |
| --- | --- |
| 账期 | 当前账单所属月份 |
| 地区 | 所在区域 |
| 原价 | 该区域产生的费用原价 |
| 代金券抵扣 | 该区域使用代金券抵扣金额 |
| 现金支付 | 实际支付金额 |

2. 按产品统计

* 按产品类型拆分账单金额。

| 字段 | 说明 |
| --- | --- |
| 账期 | 当前账单所属月份 |
| 产品名称 | 消费的产品类型 |
| 费用类型 | 消费类别 |
| 原价 | 产品在该月份产生的费用 |
| 代金券抵扣 | 产品对应优惠金额 |
| 现金支付 | 实际支付金额 |

3. 按账号统计

* 按账号展示费用分布。

| 字段 | 说明 |
| --- | --- |
| 账期 | 当前账单所属月份 |
| 用户名/ID | 消费操作来源账号 |
| 原价 | 该账号对应的费用部分 |
| 代金券抵扣 | 优惠抵扣金额 |
| 现金支付 | 实际支付金额 |

## 导出账单

点击页面右上角的 **导出** 按钮，可以导出筛选账期内账单数据，并按区域、产品、账号维度分别统计。
