# 可观测性权限说明

- Admin / Kpanda Owner

- [NS Admin](../../kpanda/user-guide/permissions/permission-brief.md#ns-admin) / [NS Edit](../../kpanda/user-guide/permissions/permission-brief.md#ns-edit)
- [NS View](../../kpanda/user-guide/permissions/permission-brief.md#ns-view)
