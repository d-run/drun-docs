# vGPU

Nvidia vGPU 是 Nvidia 开发的一项专有技术，专用于数据中心中的某些 Tesla 和 Quadro 显卡。
该技术主要面向 VDI（虚拟桌面基础设施）市场，旨在让远程用户通过远程桌面解决方案进行工作。
不同的远程应用包括 CAD/CAM 软件、游戏（例如：GeForce Now）、建筑设计以及许多其他需要图形加速的程序。

该技术通过将支持的显卡资源分配给多个虚拟机（VM）来工作。与传统虚拟机相比，其优势在于提供全图形加速，而这在 CPU 光栅化中是不太可行的。
虚拟机在数据中心和 VDI 场景中非常流行，因为它们易于迁移和为员工和用户配置。远程用户可以连接到中央 VDI 服务器，在较轻的计算机（如笔记本电脑或瘦客户端）上访问自己的工作空间。
需要大量 3D 和/或计算能力的图形工作负载并不总是能在超极本和其他“轻量级”计算机上运行。

这对于需要远程图形处理能力的办公室和其他企业环境是有意义的。但这对你意味着什么呢？
vgpu_unlock（及其变体）的目标是允许用户在性能较低的显卡或消费级显卡上运行 Nvidia vGPU 技术，
而这些显卡并非为 vGPU 设计的专业数据中心显卡。通过这种方式，用户可以将自己的显卡虚拟化给多个虚拟机使用。
例如，这可以让用户运行一个能够为朋友提供游戏功能的虚拟机，或者在 Linux 上使用 Windows 并实现图形加速（使用 Looking Glass）。
当然，还有更多的用途，但这些只是一些常见的使用案例。

## 目前支持的显卡

| vGPU 认证显卡 | GPU 芯片 | 支持的消费级等效产品 |
| :-------------: | :------: | -------------------: |
|    Tesla M10    | GM107 x4 | 大多数第一代 Maxwell 显卡 |
|    Tesla M60    | GM204 x2 | 大多数第二代 Maxwell 显卡 |
|    Tesla P40    |  GP102   | 大多数 Pascal 显卡 |
| Tesla V100 16GB |  GV100   | Titan V & Quadro GV100 |
| Quadro RTX6000  |  TU102   | 大多数 Turing 显卡 |
|    RTX A6000    |  GA102   | **无（不指望有）** |
